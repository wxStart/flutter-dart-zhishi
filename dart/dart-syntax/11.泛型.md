
### 泛型

#### 泛型是在函数、类、接口中指定宽泛数据类型的语法

+ 泛型函数
+ 泛型类
+ 泛型接口



#### 通常，在尖括号中，使用一个字母来代表类型，例如E、T、S、K和V等


结构如下
```
    返回类型 函数名<输入类型>（参数类型 参数）{

        函数体
    }

    T getData<T>(T value){

        return value;
    }


```



#### 作用：使用泛型可以减少重复代码

```
    <!-- 调用 -->
    getDate<int>(12);

    getData<String>('12');
    
```
